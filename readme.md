🚀 CODE TEAM | Smart College Hackathon

Информационный портал и рабочее пространство для участников хакатона CODE TEAM в КИГМ № 23. Проект предназначен для координации команд, ознакомления с техническим заданием и сдачи финальных решений.

📌 О проекте

Данный репозиторий содержит фронтенд-часть портала хакатона. Основная цель — автоматизировать процесс взаимодействия между организаторами и участниками, исключив дублирование функций, которые уже реализованы в существующих системах (таких как МЭШ).

Основные разделы:

Инфо-панель: Тайминг мероприятия и общие правила.

Задания (Case Study): Подробное описание кейса «Smart College».

Форма сдачи: Интегрированная система отправки результатов.

🎯 Кейс: Smart College

Задача участников — разработать мобильного помощника для первокурсника КИГМ № 23.
Внимание: Функционал «Цифровой зачетки» и оценок не требуется, так как эти данные предоставляются сервисом МЭШ.

Приоритетные направления разработки:

Интерактивный гид по столовой: Меню, график загруженности, система отзывов.

Навигатор по корпусу: Карта аудиторий, мастерских и ИТ-полигонов.

Геймификация: Система достижений за внеучебную активность.

🛠 Технический стек

Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+).

Icons: Font Awesome 6.

Design: Figma.

VCS: GitHub.

📝 Требования к сдаче работы

При отправке финального решения через портал, команда обязана предоставить:

Название команды и номер группы.

GitHub Link: Ссылка на публичный репозиторий с исходным кодом.

Figma Link: Ссылка на интерактивный прототип (убедитесь, что настроены права доступа "Can view").
