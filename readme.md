Интерактивная платформа для проведения хакатона «Технологическое Видение 2026» в колледже КИГМ №23. Проект объединяет в себе информационный лендинг, систему подачи заявок и панель управления для жюри.

🚀 Основные возможности

Информационный хаб: Главная страница с таймингом, критериями оценивания и информацией о призах.

Брифы заданий: Четыре ключевых направления разработки (Навигация, Общение, Рейтинг, Поддержка) с описанием приоритетов.

Система регистрации: Форма отправки проектов с автоматической генерацией ID заявки и синхронизацией с облачной базой данных.

Панель управления (Admin): Защищенный паролем интерфейс для жюри, позволяющий просматривать все входящие заявки, переходить по ссылкам на репозитории и удалять записи.

Современный UI: Адаптивный «стеклянный» дизайн (Glassmorphism) с глубокими цветами и анимациями.

🛠 Технологический стек

Frontend: HTML5, CSS3 (Tailwind CSS).

Icons: Lucide React & FontAwesome.

Backend/Database: Firebase Firestore (Real-time updates).

Auth: Firebase Anonymous Authentication.

Fonts: Inter & JetBrains Mono.

📦 Структура данных (Firestore)

Проект использует строго заданную структуру путей для обеспечения безопасности и корректной работы прав доступа:
artifacts/{appId}/public/data/submissions

Каждый документ заявки содержит:

team: Название команды.

group: Учебная группа.

captain: ФИО капитана.

git: Ссылка на репозиторий.

figma: Ссылка на демо/дизайн.

id: Уникальный номер заявки (формат CT-26-XXXX).

ts: Таймштамп создания.

⚙️ Установка и запуск

Поскольку проект реализован в формате Single File Application (все ресурсы в одном HTML-файле), установка не требуется:

Скачайте файл index.html.

Откройте его в любом современном браузере.

Для работы функционала отправки и администрирования требуется подключение к интернету (для связи с Firebase).

🔐 Доступ администратора

Для входа в панель управления:

Нажмите на ссылку Access_Terminal в футере сайта.

Введите код доступа: 2026.
